### US-002: Base layout with dark mode toggle
Status: in_progress
Depends on: US-001

Description:
As a user, I want a clean base layout with dark mode toggle so I can use my preferred theme.

Acceptance Criteria:
- [ ] Create App.svelte with centered container (max-width ~600px)
- [ ] Add generous padding/whitespace matching Things 3 aesthetic
- [ ] Add dark mode toggle button in top-right corner
- [ ] Toggle adds/removes 'dark' class on html element
- [ ] Persist theme preference to LocalStorage
- [ ] Load saved preference on app init
- [ ] Style both light and dark themes with Tailwind dark: variants
- [ ] Example: click toggle -> theme switches with smooth transition
- [ ] Example: refresh page -> theme persists
- [ ] Negative case: LocalStorage cleared -> defaults to light mode
