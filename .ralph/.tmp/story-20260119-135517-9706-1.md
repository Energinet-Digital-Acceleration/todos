### US-006: Complete todo interaction with animation
Status: in_progress
Depends on: US-005

Description:
As a user, I want to complete todos with satisfying animation feedback.

Acceptance Criteria:
- [ ] Click checkbox to toggle completion
- [ ] Completing: checkbox fills with subtle animation
- [ ] Completing: title gets strikethrough + fades
- [ ] Completing: item slides down to completed section after brief delay
- [ ] Use Svelte transitions (fly, fade) for animations
- [ ] Animation duration ~300ms, eased
- [ ] Example: click checkbox -> animation plays, moves to completed
- [ ] Example: rapid clicks -> state toggles correctly
- [ ] Negative case: uncompleting -> moves back to active list
