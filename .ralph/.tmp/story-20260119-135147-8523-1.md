### US-005: Active todo list display
Status: in_progress
Depends on: US-003, US-004

Description:
As a user, I want to see my active todos in a clean list.

Acceptance Criteria:
- [ ] Create TodoList.svelte component
- [ ] Display active (not completed) todos
- [ ] New todos appear at top of list
- [ ] Each todo shows: checkbox + title
- [ ] Checkbox is circular, Things 3 style
- [ ] Generous vertical spacing between items
- [ ] Hover state shows subtle background change
- [ ] Example: 3 active todos -> all 3 displayed in order
- [ ] Example: no active todos -> list area empty (no message)
- [ ] Negative case: completed todos -> not shown in active list
