### US-004: Todo input component
Status: in_progress
Depends on: US-002, US-003

Description:
As a user, I want an always-visible input at the bottom to add todos quickly.

Acceptance Criteria:
- [ ] Create TodoInput.svelte component
- [ ] Fixed position at bottom of todo list area
- [ ] Clean minimal styling matching Things 3
- [ ] Placeholder text: 'Add a todo...'
- [ ] Enter key submits and clears input
- [ ] Escape key clears input without submitting
- [ ] Empty submissions are ignored (no empty todos created)
- [ ] Subtle focus ring styling
- [ ] Example: type 'Buy milk' + Enter -> todo added, input cleared
- [ ] Example: type 'test' + Escape -> input cleared, no todo added
- [ ] Negative case: submit empty/whitespace -> nothing happens
