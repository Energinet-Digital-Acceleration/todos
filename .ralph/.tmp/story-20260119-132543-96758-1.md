### US-001: Project scaffold with Vite + Svelte + Tailwind
Status: in_progress
Depends on: None

Description:
As a developer, I want the project scaffolded so I can start building.

Acceptance Criteria:
- [ ] Run `bun create vite . --template svelte-ts` to scaffold
- [ ] Install deps: `bun add -d tailwindcss postcss autoprefixer`
- [ ] Run `bunx tailwindcss init -p` to create config files
- [ ] Configure tailwind.config.js content paths for Svelte
- [ ] Add Tailwind directives to src/app.css
- [ ] Install eslint + prettier: `bun add -d eslint prettier eslint-plugin-svelte prettier-plugin-svelte`
- [ ] Add scripts to package.json: build, dev, lint, check
- [ ] Example: `bun run dev` starts dev server
- [ ] Example: `bun run build` completes without errors
- [ ] Negative case: missing tailwind config -> build fails
